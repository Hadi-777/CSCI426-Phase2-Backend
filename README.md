# CSCI426 - Advanced Web Programming (Phase 2 Backend)

## Project Description
This repository contains the **Node.js backend** for the **Dr.Online** web application.  
It provides **user authentication (Signup/Login)** and **CRUD APIs** connected to a **MySQL** database, enabling the frontend to manage application data securely.

## Technologies Used
- Node.js
- Express.js
- MySQL
- JWT Authentication
- dotenv
- bcryptjs
